Passos para execução do proxy utilizando navegador firefox:

1- Com a pasta contendo os códigos já extraída, abra o navegador firefox.
2- Vá em opções, preferências
3- Avançado, Internet, Proxy de rede, configurar conexão
4- Selecionar configuração manual de proxy
5- Em proxy HTTP escrever localhost, e em porta 54321
6- Selecionar opção Usar este proxy para todos os protocolos
7- Após feito isso, entrar na pasta extraída inicialmente contendo os códigos, e rodar o principal com o comando "python main.py"
8- Feito isso o navegador estará utilizando o proxy com cache, onde a descrição será guardada no arquivo proxy.log
